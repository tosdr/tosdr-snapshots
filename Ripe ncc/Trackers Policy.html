Cookies used by the RIPE NCC

This page may also cover other non-cookie technologies (e.g. anonymous device fingerprinting, anonymous unique identifiers).

For more information on how we use cookies, check our Privacy Statement.

Strictly necessary cookies

Cookie name Purpose Term

crowd.token_key

This cookie identifies the user. It is used to authenticate logged-in single-sign-on (RIPE NCC Access) users.

Expires when the browser session is closed

crowd.ripe.hint

This cookie is used to identify logged-in users on RIPE NCC services (Single Sign On).

Expires when the browser session is closed

PHPSESSID

This is a session cookie, used to identify a user's session on a website. It is commonly used in PHP-based web applications to store a unique identifier for a user's session on the server.

Expires when the browser session is closed

JSESSIONID

This is a general-purpose platform session cookie, used by sites written in JSP. Usually used to maintain an anonymous user session by the server.

Expires when the browser session is closed

*_sessionid

This cookie is used to authenticate a user's logged-in status and permissions.

Expires after 2 weeks

*_csrftoken

This cookie is associated with the Django web development platform for Python. It is designed to help protect a site against a particular type of software attack on web forms.

Expires after 1 year

MoodleSession

You must allow this cookie used by our learning platform Moodle into your browser to provide continuity and maintain your login from page to page.

Expires when the browser session is closed

__cf_bm

The __cf_bm cookie is a cookie necessary to support Cloudflare Bot Management, currently in private beta. As part of our bot management service, this cookie helps manage incoming traffic that matches criteria associated with bots.

Expires after 30 minutes

cf_chl_cc_xxxxxxxxxxxx

These cookies are used by Cloudflare for the execution of Javascript or Captcha challenges. They are not used for tracking or beyond the scope of the challenge.

Expires when the browser session is closed

cf_chl_2

These cookies are used by Cloudflare for the execution of Javascript or Captcha challenges. They are not used for tracking or beyond the scope of the challenge.

Expires after a few seconds

AWSALB

AWS ELB application load balancer.

Expires after 1 week

AWSALBCORS

This cookie is managed by AWS and is used for load balancing.

Expires after 1 week

activeMembershipId

This cookie represents the ID of the member currently being edited.

Expires when the browser session is closed

activeRegId

Holds the currently selected RegID.

Expires when the browser session is closed

nccmeeting

Used for sticking the user session to a particular application backend.

Expires when the browser session is closed

.AspNetCore.Antiforgery.*

This cookie is a security token for application operation generated by the  Microsoft.net framework.

Expires when the browser session is closed

arlo_cid

Arlo_cid is the cart ID and is required for the checkout to function.

Expires after 30 minutes 

_GRECAPTCHA

Set by Google reCAPTCHA, which protects forms against spam enquiries.

Expires after 6 months

areYourCookiesEnabled

Used to check if the user's browser supports cookies.

Expires when the browser session is closed

mtm-paq

This is our cookie banner, where you can accept or reject cookies.

Expires after 1 year

tag-consent-*

This is the cookie banner we use for embedded content where you accept cookies per provider of the embedded content.

Expires after 1 month

websiteSearch

Added when the user selects the website search as the default NCC search.

Expires after 1 year

dnsmon_dnsmon_color_range_pls

Stores user's preferences in the DNSMON widget.

Expires after 4 months

pref-ui-mode

Keeps information about the selected editor type, i.e. single-line editor or text editor.

Expires after 1 day

zlcprivacy

Used by the Zendesk chat widget to store the visitor's decision on cookie acceptance.

Expires after 1 year

zlcmid

Used by the Zendesk chat widget to store the visitor's ID for widget authentication.

Expires after 1 year

open-live-chat

Used to open the Zendesk chat widget window. 

Expires when the browser session is closed

_pk_testcookie_domain

Matomo cookie that determines whether the browser accepts cookies. 

Created, then immediately deleted

cookies-accepted

This is a cookie that we used to identify that a visitor accepted our previous Cookie Policy. It was set for a period of one year. We have now deprecated it, and therefore it can be removed when seen.

Expires after 1 year

AUTH_SESSION_ID

This cookie is used to identify logged in users among RIPE NCC services (Single Sign On).

Expires when the browser session is closed

AUTH_SESSION_ID_LEGACY

This cookie is used to identify logged in users among RIPE NCC services (Single Sign On).

Expires when the browser session is closed

KC_RESTART

This cookie is used to identify logged in users among RIPE NCC services (Single Sign On).

Expires when the browser session is closed

KEYCLOAK_IDENTITY

This cookie is used to identify logged in users among RIPE NCC services (Single Sign On).

Expires when the browser session is closed

KEYCLOAK_IDENTITY_LEGACY

This cookie is used to identify logged in users among RIPE NCC services (Single Sign On).

Expires when the browser session is closed

KEYCLOAK_SESSION

This cookie is used to identify logged in users among RIPE NCC services (Single Sign On).

Expires after 8 hours

KEYCLOAK_SESSION_LEGACY

This cookie is used to identify logged in users among RIPE NCC services (Single Sign On).

Expires after 8 hours

Performance cookies

_pk_ref

These cookies are created and set by Matomo, a third-party application, to help us improve the performance of our website. The data collected by this cookie is anonymous, stored internally within the RIPE NCC and is not shared with any third party.

Expires after 6 months

_pk_cvar

These cookies are created and set by Matomo, a third-party application, to help us improve the performance of our website. The data collected by this cookie is anonymous, stored internally within the RIPE NCC and is not shared with any third party.

Expires after 30 minutes

_pk_id

These cookies are created and set by Matomo, a third-party application, to help us improve the performance of our website. The data collected by this cookie is anonymous, stored internally within the RIPE NCC and is not shared with any third party.

Expires after 13 months

_pk_ses

These cookies are created and set by Matomo, a third-party application, to help us improve the performance of our website. The data collected by this cookie is anonymous, stored internally within the RIPE NCC and is not shared with any third party.

Expires after 30 minutes

_pk_hsr

These cookies are created and set by Matomo, a third-party application, to help us improve the performance of our website. The data collected by this cookie is anonymous, stored internally within the RIPE NCC and is not shared with any third party.

Expires after 30 minutes
