    Our website uses cookies in order to improve your user experience.

    A cookie is a file that contains an identifier (a sequence of letters and numbers) that is sent by a web server to a web browser and is stored by the browser. The identifier is then sent back to the server each time the browser requests a page from the server.

    The cookies used in this website do not contain any information that personally identifies a user, only saving generic information.
