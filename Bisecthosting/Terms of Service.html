Minecraft Servers

Support for Minecraft Services

  • All Minecraft services that we provide are unmanaged. We do not offer plugin or mod installation for servers unless the premium support addon is purchased (See Premium Support Section).
  • RAM is restricted based on plan ordered. Your server will be allocated with the full amount you purchased. Any memory used for additional java flags (Such as PermGen) will use this allocated memory therefore actual memory in game may not show the full amount that your server is allocated.
  • Slots on our minecraft servers are unrestricted on Premium plans however are restricted on budget plans based on the amount stated when purchasing unless the unlimited slots addon is purchased
  • Unlimited slots on servers means you can set your server to any slot number you want, this does not mean your server will be able to handle unlimited players. The amount of players your server can handle depends on the RAM you have and plugins/mods/settings on your server.
  • CPU intensive servers may be suspended without warning to prevent causing any issues to other clients. You will be notified of this and will give you as much advice possible to get your server using a reasonable amount of CPU resources.
  • Disk space for budget and premium servers is unlimited based on fair-usage for the node you are on.
  • Please note that we do not allow the use of any automated backup plugins on Minecraft servers, as these plugins cause massive disk usage which can result in negatively affecting other customers' services.
  • If you are found to be using a script that increases the RAM allocation of your server to more than purchased your server will be terminated without refund.
  • Plugins that allow multiple servers to be run on the same package are not allowed. Examples of this are the subservers spigot/bukkit plugin.

Storage

If you go over this limit, you will be warned, if no response is made, we may be forced to delete your data to avoid the overage affecting other users. Backups

  • Complimentary backups are only provided for users on the Premium Shared Minecraft plans
  • BisectHosting provides 7 day backups where each day a backup is taken between 3am and 10am local server time.
  • Backups are held on a separate storage drive which is not accessible by the customer
  • If backups are not available due to an error, BisectHosting is not responsible for the loss of your file(s), content, worlds etc.
  • Backups can be requested in the following forms:
  • Download link - The backup can be zipped up and downloaded from our servers
  • Restored to server - Backup can be restored to the customers server, all existing files will be overwritten unless otherwise stated.

Backups ignore the following files:

  • Dynmap
  • mcmmo_data
  • .log files and anything ending in .log i.e. log.gz
  • .tar
  • Backup/MineBackup folders
  • .sql
  • Block logging data such as CoreProtect
  • .zip files in the root directory
