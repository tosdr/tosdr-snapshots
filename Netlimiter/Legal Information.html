How is user-sensitive data handled by NetLimiter

  • From our e-commerce provider, we receive general personal information provided by the user: name, e-mail address etc. We don't have access to secure payment information like credit card number.
  • The personal information like name and e-mail address is stored securely on our servers and is used solely for the purposes of user license and registration data management so we can provide support to our users like resending lost registration data. We do not provide the information to any other subject. The user database is encrypted and is not directly exposed to the Internet.
  • No user data are sent and stored out of the machine with installed NetLimiter. The program doesn't use a cloud or other means of online storage.
  • NetLimiter connects to internet only while checking for updates or registering user's license or contacting VirusTotal™ for file checks (if used). No user sensitive data are sent out off the machine.
